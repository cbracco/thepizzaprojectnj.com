# thepizzaprojectnj.com

The source code for the thepizzaprojectnj.com website.

-   Eleventy
-   PostCSS (postcss-preset-env, postcss-import, autoprefixer, cssnano)
-   ESLint
-   Stylelint
-   Prettier

# Installation

```shell
npm install
```

# Dev

```shell
npm start
```

# Lint

```shell
npm run lint
```

# Format

```shell
npm run format
```
